# ospr

[![Build Status](https://travis-ci.org/mkarpisek/orb.svg?branch=master)](https://travis-ci.org/mkarpisek/ospr)

Office 365 Sharepoint File Reporting Tool

## Getting Started

Build and package as one big uberjar:
```
mvn clean package
```

### Prerequisites

Java 8
